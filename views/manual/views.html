JaffaJS Manual

Views

View files in JAffaJS should only contain HTML code where possible.

These files are loaded usig the Loader and sit in the <div id="ajax_content"> on the index.html page

By default all view files should be wrapped in a <div role="page" > , contain a <header> tag and display all main content inside the <section class="main" >

Controllers should take care of all the dynamic and interactive functionality through standard jQuery.
