Config object.

The config object can be found indie the js/core/configs.js file and site above the DBSeeder.

  • "viewRoute" : "views/" - this is where you can specify where you would like to keep all your view files. Views are kept in the views folder by default.
  • "controllerRoute" : "js/controllers" - this is where you can specify where you keep all of the controller files. by defualt they are kept in the js/controllers/ folder.
  • "initPage" : "main.html" - in the Loader there is an init function, that loads the first page on app load. This page can be specified here. The function uses teh viewroute varaible above to determine where the view is. By default teh initPage is main.html
  • DBname - this is where you can specifiy the name of your SQLite database if you would like one, by default there is no db.
  • DBversion - this is the version of SQLite you would liek to use, but default we use version 1.0 but this can be changed. (please make sure teh version you are using is supportred by phonegap.)
  • DBdisplayName - a human name for the db.
  • DBsize - the size in bytes of your database.
  • MenuType: fade this how the jaffascript menu will fade in or out, this is fade by default but currently has the ability to be slide if needed too.
  • Months

    JaffaJS makes it as easy as possible to support language and calender variations wjile still trying to maintain a lightweight framework. We have placed a standard Gregorian calender in English into this section by default but this can be changed if neccessary. Then all time based events you call can go directly or indirectly (through the Time helper)

    • name
    • shortName
    • amount - amoutnt of days in each month from jan - dec
    • amoutnLeapYear - amount of days in each month in a leap year from jan - dec

these variable can be called by using configs.desiredVariable i.e configs.initPage or for the nested vars cofigs.months.shortName[5] if you would like to return the value of "may"

you are free to add any global variable you may need into this object.
