JaffaJS Manual

Core

The core files sit inside the the folder js/core/ and contain the following files:

  1. configs.js - this is where you can set global configerations. more about this file can be found here
  2. database.js - this is the class that handles database calls to SQLite files in phonegap. more about this can be found here
  3. init.js - This is where inital calls to the controller are run. You are also free to add whatever calls to otehr files in here
  4. jaffaScript.js - this is used for rendering menus. more can be found here
  5. jquery-1.9.1.min.js - is the current build of jQuery, this is here so that the app can function without internet connectivity. IT is advised to check to make sure you have the most current stable version at all times, dependant on your build options.
  6. loader.js - this is where teh pages and/or resources are loaded. more can be found here
  7. localstorage.js - this is a file that handle HTML5 localstorage transaction. more cn be found here
  8. main.js - this is where the time, data type and AJAX helper objects are kept.

All files here are called from the file index.html in between the CORE comments. If you need to add any further core classes, the make sure they are placed here inside <script></script> tags
