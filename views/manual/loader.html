The Loader

The loader is the class that allows pages to be loaded easily using jQueries .load() class.

The loader has a haschange event listener so you can just throw urls starting with # and the file route into an anchor tag and the loader will load the page, provided the page sits inside the views file.

this base views route can be altered inside the /js/core/configs.js file. More on this file can be found here

alternately if you would like to could force a page by calling loader.loadNewPage(pageName) if oyu would like a whole page change, or loader.loadResource(pageName , element) if you would like to just place a segment of a page into an existing page.

loader.loadNewPage(pageName) where pageName is the route to the page insde the views file.

loader.loadResource(pageName , element) where the pageName is the route to a segment that sits inside the views file by default (but can be cahnged as mentioned above) , and teh element is the class or id of the element that needs to be loaded. element needs to a be string looking like '#element' for id or '.element' for class etc. This will then change add your desired content into the container you have specified using a very raw and unclean .load() function.

loader.goBack() : this function can be added to any back button to force a return to the previous page.

feel free to customise these functions for your needs.
